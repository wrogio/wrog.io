# wrog.io
wrog.io sources

## Installation

First compile wrog.io using gcc [gcc compiler instructions](https://man7.org/linux/man-pages/man1/gcc.1.html)

```bash
make install
```

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
